The `Lovely` module handles downloading, installing, and managing the Lovely injector framework that enables Balatro mods to work. It provides platform-specific installation, version checking, and injector enable/disable functionality.

## Overview

Lovely is the mod injection framework required for Balatro mods. This module provides:

- **Installation**: Downloads and installs platform-specific Lovely binaries
- **Version Management**: Checks for updates with 24-hour caching
- **Toggle Control**: Enable/disable injector without uninstalling
- **Platform Support**: Windows DLL, macOS dylib, Linux .so

## Platform-Specific Details

<ParamField path="Windows/Proton" type="version.dll">
  Uses DLL injection via `version.dll` proxy
</ParamField>

<ParamField path="macOS" type="liblovely.dylib">
  Uses `DYLD_INSERT_LIBRARIES` environment variable
</ParamField>

<ParamField path="Linux (native)" type="liblovely.so">
  Uses `LD_PRELOAD` environment variable
</ParamField>

### get_latest_lovely_version()

Queries GitHub for the latest Lovely release version.

```rust
pub async fn get_latest_lovely_version() -> Result<String, AppError>
```

<ResponseField name="Returns" type="String">
  Version string (e.g., "0.8.0") without 'v' prefix
</ResponseField>

**Caching Strategy**:
- **In-Memory**: 24 hours
- **Disk**: Survives app restarts (stored in `lovely_version_cache.json`)
- **Network Timeout**: 10 seconds

**Implementation** (lovely.rs:444-530):
```rust
// Check in-memory cache
if let Some(cached) = guard.as_ref()
    && cached.fetched_at.elapsed() < Duration::from_secs(VERSION_CACHE_TTL_SECS)
{
    return Ok(cached.version.clone());
}

// Check disk cache (survives restarts)
if let Some(version) = load_persisted_version_cache() {
    return Ok(version);
}

// Fetch from GitHub
let resp = client
    .get("https://github.com/ethangreen-dev/lovely-injector/releases/latest")
    .send()
    .await?;

let location = resp.headers().get(LOCATION);
let version = location.split('/').last().trim_start_matches('v');
```

<Note>
GitHub redirects `/releases/latest` to `/releases/tag/vX.Y.Z`. The version is extracted from the redirect URL.
</Note>

**Example**:

```rust
use bmm_lib::lovely::get_latest_lovely_version;

let version = get_latest_lovely_version().await?;
println!("Latest Lovely version: {}", version);
```

## Injector Toggle Functions

### set_injector_enabled()

Enables or disables the Lovely injector by renaming files.

```rust
pub fn set_injector_enabled(enabled: bool) -> Result<(), AppError>
```

<ParamField path="enabled" type="bool" required>
  `true` to enable, `false` to disable
</ParamField>

**Behavior**:
- **Enable**: Renames `.disabled` files back to active names
- **Disable**: Renames active files to `.disabled`
- **Preserves Files**: Files are renamed, not deleted

**File Mappings**:

| Platform | Active | Disabled |
|----------|--------|----------|
| macOS | `liblovely.dylib` | `liblovely.dylib.disabled` |
| Windows/Linux | `version.dll` | `version.dll.disabled` |
| Linux (native) | `liblovely.so` | `liblovely.so.disabled` |

**Example**:

```rust
use bmm_lib::lovely::set_injector_enabled;
use bmm_lib::database::Database;

// Disable mods temporarily
set_injector_enabled(false)?;

// Save preference
let db = Database::new()?;
db.set_launch_mode("vanilla")?;
```

## Version Cache Implementation

### In-Memory Cache

```rust
struct CachedVersion {
    version: String,
    fetched_at: Instant,
}

static VERSION_CACHE: OnceLock<Mutex<Option<CachedVersion>>> = OnceLock::new();
const VERSION_CACHE_TTL_SECS: u64 = 60 * 60 * 24; // 24 hours
```

### Disk-Persisted Cache

**Structure** (lovely.rs:38-43):
```rust
#[derive(serde::Serialize, serde::Deserialize)]
struct PersistedVersionCache {
    version: String,
    fetched_at_unix: u64,  // Unix timestamp in seconds
}
```

**Storage Location**:
```
~/.config/Balatro/lovely_version_cache.json
```

**Save** (lovely.rs:75-95):
```rust
fn save_persisted_version_cache(version: &str) {
    let now = SystemTime::now().duration_since(UNIX_EPOCH)?.as_secs();
    let cached = PersistedVersionCache {
        version: version.to_string(),
        fetched_at_unix: now,
    };
    fs::write(&path, serde_json::to_string(&cached)?)?;
}
```

**Load** (lovely.rs:55-72):
```rust
fn load_persisted_version_cache() -> Option<String> {
    let cached: PersistedVersionCache = serde_json::from_str(&content).ok()?;
    let now = SystemTime::now().duration_since(UNIX_EPOCH).ok()?.as_secs();
    
    if now.saturating_sub(cached.fetched_at_unix) < VERSION_CACHE_TTL_SECS {
        Some(cached.version)
    } else {
        None
    }
}
```

**Enable Logic** (lovely.rs:702-719):
```rust
if disabled_path.exists() {
    if active_path.exists() {
        // Corrupt state: both exist, remove disabled
        fs::remove_file(disabled_path)?;
    } else {
        // Normal: rename disabled to active
        fs::rename(disabled_path, active_path)?;
    }
}
```

**Disable Logic** (lovely.rs:721-738):
```rust
if active_path.exists() {
    if disabled_path.exists() {
        // Corrupt state: both exist, remove active
        fs::remove_file(active_path)?;
    } else {
        // Normal: rename active to disabled
        fs::rename(active_path, disabled_path)?;
    }
}
```

## Error Handling

Common errors returned:

- `AppError::Network` - Download failure or timeout
- `AppError::DirNotFound` - Config directory or Balatro installation not found
- `AppError::DirCreate` - Failed to create bins directory
- `AppError::FileCopy` - Failed to copy cached files
- `AppError::FileWrite` / `AppError::FileRead` - I/O errors
- `AppError::InvalidState` - Version parsing failure, extraction error
- `AppError::UnsupportedArchitecture` - Unknown macOS architecture
- `AppError::SystemDetection` - Failed to detect system information

## Examples

### Installing Lovely

```rust
use bmm_lib::lovely::ensure_lovely_exists;
use bmm_lib::database::Database;

let lovely_path = ensure_lovely_exists().await?;
let version = get_latest_lovely_version().await?;

// Save version to database
let db = Database::new()?;
db.set_lovely_version(&version)?;

println!("Lovely {} installed at {}", version, lovely_path.display());
```

### Checking for Updates

```rust
use bmm_lib::lovely::get_latest_lovely_version;
use bmm_lib::database::Database;

let db = Database::new()?;
let installed_version = db.get_lovely_version()?.unwrap_or_default();
let latest_version = get_latest_lovely_version().await?;

if installed_version != latest_version {
    println!("Update available: {} -> {}", installed_version, latest_version);
}
```

### Toggling Vanilla Mode

```rust
use bmm_lib::lovely::{set_injector_enabled, is_injector_enabled};
use bmm_lib::database::Database;

let db = Database::new()?;
let vanilla_mode = db.get_launch_mode()? == "vanilla";

set_injector_enabled(!vanilla_mode)?;
println!("Mods {}", if vanilla_mode { "disabled" } else { "enabled" });
```

### Complete Reinstall

```rust
use bmm_lib::lovely::{remove_installed_lovely, ensure_lovely_exists};

// Remove old installation
remove_installed_lovely().await?;

// Install fresh copy
let lovely_path = ensure_lovely_exists().await?;
println!("Lovely reinstalled successfully");
```
